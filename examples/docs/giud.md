# xxw-mobile-design 

`xxw-mobile-design ` 是一款基于 `Vant` 的前端 UI 组件库，主要用于快速开发 移动端 项目
[演示地址](https://vv-ui.github.io/VV-UI)

 

## 特性

- 基于 `Vant` 开发的 UI 组件
- 封装常用的业务组件
